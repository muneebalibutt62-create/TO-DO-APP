# To-Do App Pro (Level 3)

A modern desktop productivity application built with Python and Tkinter.  
This app helps users manage tasks efficiently with priorities, categories, search, and progress tracking.

---

## 🚀 Features

- ➕ Add tasks with priority levels
- 📂 Task categories (Work / Study / Personal)
- ✔ Mark tasks as completed (double-click)
- 🗑 Delete tasks easily
- 🔍 Search tasks in real time
- 📊 Progress tracking (% completion)
- 🌙 Dark mode UI
- 💾 Auto-save using JSON

---

## 🧠 How It Works

Tasks are stored in a local JSON file.  
Each task contains:

- Task text
- Priority (High / Medium / Low)
- Category
- Completion status
- Due date (optional)

The app updates the UI dynamically based on user actions.

---

## 🛠️ Technologies Used

- Python
- Tkinter (GUI)
- JSON (Data Storage)
- datetime module
